# Primer-trabajo-Mongo
Este repositorio contiene los archivos y el desarrollo del taller práctico sobre el uso de **MongoDB**, centrado en la creación de conexiones y la persistencia de datos.

## Descripción del Proyecto
El objetivo de esta actividad fue comprender el flujo de trabajo básico con bases de datos NoSQL, desde la configuración inicial en la nube hasta la integración con entornos locales y herramientas de análisis como Google Colab.

### Hitos del trabajo:
* **Configuración en la nube:** Creación de la instancia en línea, gestión de claves y obtención de la cadena de conexión necesaria.
* **Integración local:** Configuración de la conexión entre el cliente instalado en el computador y MongoDB en la nube.
* **Desarrollo en Google Colab:** Implementación de un script en Python para capturar datos de estudiantes (nombre, edad, carrera) y gestionar su almacenamiento.
* **Persistencia:** Verificación del funcionamiento de la base de datos al validar que, tras recargar la aplicación, los registros previamente ingresados se mantenían correctamente.
* **Exportación:** Generación y descarga del archivo `.csv` con los datos consolidados.

## Reflexión personal
Fue una experiencia de aprendizaje muy enriquecedora. A pesar de enfrentar desafíos técnicos iniciales —como errores en la generación del *link* de conexión que requirieron volver a configurar los parámetros—, logré solucionar los inconvenientes y comprender el comportamiento de la herramienta. Resultó fascinante observar cómo la base de datos interactúa en tiempo real y persiste la información entre diferentes sesiones de recarga.

## Archivos incluidos
